Тест кейс №1
Название тест кейса: Поиск в гитхаб возвращает ссылку на найденный репозиторий

Предусловия тест кейса: Открыт сайт https://github.com

Шаги тест кейса:
1 шаг: В поле поиска введено одно из значенений {"Dydkina/qaGuru"; "eroshenkoam/allure-example"}
2 шаг: Выполнено нажатие на кнопку "ENTER"

Ожидаемый результат:
Отображена страница результата поиска.
Отображено кликабельное название репозитория.
Название репозитория соответствует введенному в полле поиска

Тест кейс №2
Название тест кейса: Поиск в соннике возвращает толкование, содержащее в заголовке искомое слово

Предусловия тест кейса: Открыт сайт https://www.sonnik-online.net

Шаги тест кейса:
1 шаг: В поле поиска введено одно из значенений {ДОМ; РЕКА; ПТИЦА}
2 шаг: В выпадающем списке результатов выбрано первое значение

Ожидаемый результат:
Отображена страница результата поиска.
Отображен заголовок толкования, который содержит слово, введенное в поиске
